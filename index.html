Navy Electricity and Electronics Training Series in HTML

This site hosts a conversion of the Navy Electricity and Electronics Training Series (NEETS) from PDF to HTML format.

The following process was used to generate these pages:

  • Use Apache TIKA to extract text and embedded files from the PDF
  • Use ImageMagick to convert TIF to PNG
  • Create a domain model in f#, use HTMLAgilityPack to traverse "ugly" HTML and create json model
  • Use json model to generate "pretty" HTML

The above steps were accomplished in a Beaker Notebook. The output is mostly correct with some errors.
